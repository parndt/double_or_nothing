# Double Or Nothing

Figure out the exact date when somebody will be twice your age.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'double_or_nothing'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install double_or_nothing
